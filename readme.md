## :art: 介绍

羊宝云是一款`根图床`WEB程序。用户可以向Cloud_Yang中上传任意文件，并返回链接

Cloud_Yang会将用户上传的图片保存到`本地`，而非将图片上传到`其它公共容器`中(例如七牛、新浪等)，满足`搭建一个提供图床存储、读取服务`的需求。

## :sparkles: 功能

- 拖拽文件，粘贴图片上传，并返回图片链接
- 没有太多杂乱功能，仅仅是一个图床/云盘
- 访问图片会展示图片，访问其他文件则会下载
